# linear-and-logistic-models
This repository explores the relationship between vehicle characteristics and environmental impact using the **1995-2014 Fuel Consumption Ratings** dataset from the [Open Government of Canada](https://open.canada.ca/data/en/dataset/98f1a129-f628-4ce4-b24d-6f16bf24dd64). By analyzing features such as **Fuel Type**, **Engine Size**, and **Cylinders**, I implement three core machine learning models to predict performance: **Simple Linear Regression** is used to model the direct correlation between fuel consumption and **CO2 Emissions**, **Multiple Linear Regression** incorporates multiple technical specifications to provide a more comprehensive emission forecast, and **Logistic Regression** is applied as a classification tool to categorize vehicles as high or low emitters based on their efficiency ratings. This project demonstrates a end-to-end data science workflow—from cleaning real-world government data to evaluating predictive accuracy—offering a practical look at how statistical learning can be used for environmental analysis.
